# Thirtyfold Studio — Brand Guidelines & Style Guide

**Version:** 1.0  
**Prepared for:** Thirtyfold Studio  
**Primary logo reference:** Bold layered 3D wordmark, “THIRTYFOLD” with “STUDIO” badge  
**Brand category:** AI studio / creative automation agency / digital growth systems

---

## 1. Brand Essence

### 1.1 Brand Name

**Thirtyfold Studio**

The name carries a growth and multiplication idea: output that compounds. It also has a biblical reference from the parable of the sower, where good seed produces fruit “thirtyfold, sixtyfold, and a hundredfold.” The brand should therefore avoid empty hype. It should communicate disciplined multiplication, useful output, and measurable fruit.

### 1.2 Brand Idea

**Thirtyfold Studio builds AI-powered systems that multiply a business’s output.**

The visual identity should feel:

- Creative, but not childish
- Technical, but not cold
- Bold, but not messy
- Energetic, but still professional
- Memorable, but still usable in business contexts

### 1.3 Brand Personality

| Trait | Meaning |
|---|---|
| **Bold** | The brand is confident and visible. |
| **Systematic** | The work is built around repeatable processes, automation, and measurable outcomes. |
| **Creative** | The studio has visual taste and production energy. |
| **Useful** | The brand does not sell vague AI. It sells practical business leverage. |
| **Multiplying** | The brand promise is compounding output: more leads, more reviews, more follow-up, more content, more systems. |

### 1.4 Positioning Statement

**Thirtyfold Studio is an AI automation and creative systems studio helping businesses turn existing attention, leads, reviews, and workflows into multiplied output through practical AI agents and automation.**

## 2. Logo System

## 2.1 Primary Logo

The primary logo is a custom-style wordmark composed of:

1. **Main wordmark:** `THIRTYFOLD`
2. **Descriptor badge:** `STUDIO`

The logo uses a bold uppercase geometric block style with a black main face and colorful offset layers behind the letters.

### Visual Structure

```text
THIRTYFOLD
   STUDIO
```

The `THIRTYFOLD` wordmark is dominant. `STUDIO` is secondary and should never visually compete with the main name.

---

## 2.2 Logo Style Description

The logo style is:

**Bold geometric 3D wordmark with colorful hard-edged offset layers.**

It combines:

- Heavy uppercase display typography
- Solid black face lettering
- Bright colored offset shadows
- Flat vector-like geometry
- Retro pop-art influence
- Digital studio energy
- High contrast and strong memorability

The mark should look crisp, sharp, and constructed. It should not look glossy, metallic, handwritten, soft, or cartoonish.

---

## 2.3 Logo Construction

### Main Wordmark

**Text:** `THIRTYFOLD`  
**Case:** Uppercase only  
**Weight:** Extra-bold / black display weight  
**Style:** Blocky geometric display sans-serif  
**Face color:** Near-black charcoal  
**Effect:** Offset colored shadow layers

### Descriptor Badge

**Text:** `STUDIO`  
**Case:** Uppercase only  
**Style:** Clean sans-serif  
**Tracking:** Wide  
**Placement:** Centered below `THIRTYFOLD`  
**Container:** Solid black rectangular badge  
**Text color:** White

---

## 2.4 Logo Color Anatomy

The logo uses a dark primary letter face with bright secondary depth layers.

### Main Face

| Element | Color | Hex |
|---|---:|---:|
| Main letter face | Charcoal black | `#1E1D1D` |
| Badge background | Charcoal black | `#1E1D1D` |
| Badge text | White | `#FFFFFF` |

### Accent Shadow Palette

| Accent | Hex | Usage |
|---|---:|---|
| Bright Green | `#20E16F` | Letter shadow / energetic growth accent |
| Cyan Blue | `#03B9FB` | Letter shadow / digital tech accent |
| Aqua Teal | `#03C3B7` | Letter shadow / modern AI accent |
| Amber Yellow | `#FCAB05` | Letter shadow / warmth and creativity |
| Purple | `#9634FB` | Letter shadow / imagination and digital edge |

---

## 2.5 Logo Proportions

Recommended approximate proportions:

| Logo Part | Recommended Proportion |
|---|---:|
| Main wordmark width | 100% |
| Main wordmark height | 70–80% of total logo height |
| STUDIO badge width | 22–28% of main wordmark width |
| STUDIO badge height | 18–24% of main wordmark height |
| Gap between main wordmark and badge | 8–12% of main wordmark height |

### Alignment Rules

- `THIRTYFOLD` must be horizontally centered.
- `STUDIO` must be centered beneath the main wordmark.
- The badge should sit close enough to feel attached to the logo, but not touch the colored shadow layers.
- The full logo should feel like one compact lockup.

---

## 2.6 Spacing and Tracking

### THIRTYFOLD

The main wordmark should have tight spacing.

Recommended feel:

- Tight but readable
- No letter collisions
- No accidental visual gaps
- No stray colored nudges or misaligned fragments
- Shadows should support the letters, not create noise

Approximate tracking direction:

```text
Tracking: -2% to -5%
```

### STUDIO

The descriptor should use wide letter spacing.

Recommended feel:

- Clean
- Balanced
- More refined than the main wordmark
- Enough spacing to read clearly at smaller sizes

Approximate tracking direction:

```text
Tracking: +20% to +40%
```

---

## 2.7 Clear Space

Always leave clear space around the logo.

Use the height of the `STUDIO` badge as the minimum spacing unit.

```text
Minimum clear space = 1x STUDIO badge height
Preferred clear space = 2x STUDIO badge height
```

Do not place text, icons, borders, or other design elements inside this clear space.

---

## 2.8 Minimum Size

Because the logo has detailed colored layers, do not use it too small.

| Use Case | Minimum Width |
|---|---:|
| Website header | 180 px |
| Social media post | 240 px |
| Presentation slide | 220 px |
| Print material | 45 mm |
| Business card | 35 mm, but simplified version preferred |
| Favicon / profile icon | Do not use full wordmark |

For small sizes, use a simplified mark such as `TF`, `30F`, or a dedicated icon.

## 3.3 Monochrome Black Logo

Use when color reproduction is limited.

- Internal documents

## 3.4 Monochrome White Logo

Use on dark backgrounds where the full-color version becomes too noisy.

**Use on:**

- Dark banners

- Presentation title slides
- Website hero sections

---

## 3.5 Simplified Icon Mark

A simplified mark is required because the full wordmark is too wide for many modern use cases.

Recommended directions:

### Option A — `TF`

A compact blocky `TF` using the same black face and colored offset layers.

### Option B — `30F`

A more ownable abbreviation connecting directly to “Thirtyfold.”

### Option C — Layered `T`

A single bold `T` with the same colorful shadow effect.

### Option D — Owl Mark

A separate owl mascot mark may be used as a secondary brand asset, but it should not replace the wordmark unless the brand system is intentionally expanded.

Recommended primary icon choice:

```text
30F
```

Reason: it is short, memorable, and directly connected to the name.

## 4. Logo Usage Rules

## 4.1 Correct Usage

Use the logo:

- On clean backgrounds
- With enough space
- At readable sizes
- In approved colors
- With the `STUDIO` badge centered
- With sharp, clean, hard-edged shadows
- As a transparent PNG or SVG

---

## 4.2 Incorrect Usage

Do not:

- Stretch or compress the logo
- Recolor the main face randomly
- Add gradients to the letter face
- Add glow effects
- Add drop shadows outside the existing 3D style
- Rotate the logo
- Place it on busy photos without a dark/light overlay
- Use low-resolution screenshots as final assets
- Let colored shadow layers create stray “nudges”
- Use the full logo as a tiny favicon
- Change `STUDIO` into another word inside the badge
- Remove the `STUDIO` badge in formal brand applications unless using an approved simplified variant

---

## 4.3 Background Control

### Best Backgrounds

| Background | Recommended Logo |
|---|---|
| White | Primary full-color |
| Light grey | Primary full-color |
| Cream | Primary full-color |
| Charcoal | Negative full-color or white mono |
| Dark forest green | Negative full-color or white mono |
| Black | Negative full-color or white mono |
| Clean photo | White mono or negative full-color |
| Busy photo | Use overlay first, then logo |

### Avoid

- High-saturation backgrounds
- Neon gradients
- Patterned backgrounds
- Cluttered imagery
- Low-contrast placement

## 5. Typography System

The exact logo lettering should be treated as custom display lettering, not a standard editable font. For consistency, use supporting typefaces that do not compete with the logo.

## 5.1 Logo Font Direction

The logo resembles a custom heavy geometric display type style.

Closest font categories:

- Bold geometric display sans
- Blocky retro display type
- 3D poster lettering
- Heavy uppercase display type

Possible reference fonts for design exploration:

- Bungee
- Anton
- League Spartan
- Archivo Black
- Montserrat Black
- Impact-style block fonts, heavily customized
- Custom vector lettering inspired by retro 3D type

Final recommendation:

```text
Do not depend on a live font for the logo. Rebuild the final logo as custom vector lettering.
```

---

## 5.2 Brand Typography

### Primary Typeface

**Inter**

Use for:

- Website UI
- Proposals
- Documents
- Case studies
- Email signatures
- Landing pages
- Dashboards

Recommended weights:

| Use | Weight |
|---|---:|
| Headings | 700 / 800 |
| Subheadings | 600 |
| Body | 400 / 500 |
| Captions | 400 |
| Buttons | 600 / 700 |

### Secondary Typeface

**Space Grotesk**

Use for:

- Strong headings

- Stickers

## 7. Visual Style

## 7.1 Overall Design Direction

Thirtyfold Studio should use a design system that feels:

- High-contrast
- Structured
- Modular
- Digital-first
- Editorial but practical
- Sharp and clean
- Slightly playful through accent color

The logo already carries strong visual energy. Supporting materials should be cleaner and more restrained.

---

## 7.2 Shapes

Use:

- Rectangles
- Modular cards
- Layered blocks
- Offset panels
- Hard-edged geometry
- Simple line icons
- Subtle grid systems

Avoid:

- Overly rounded bubble shapes
- Cartoon blobs
- Excessive glassmorphism
- Random decorative squiggles
- Overly soft startup gradients

---

## 7.3 Layout Style

Recommended layout principles:

- Strong grid
- Clear hierarchy
- Generous whitespace
- Bold headings
- Modular cards
- One main message per section
- Use color as emphasis, not decoration

---

## 7.4 Graphic Motifs

Use visual motifs based on multiplication and systems:

- Layered offset blocks
- Stacked cards
- Signal lines
- Agent nodes
- Workflow arrows
- Repeating grid dots
- Input → process → output diagrams
- “30x” repetition patterns
- Seed/growth concept used subtly, not literally everywhere

---

## 8. Photography & Imagery

## 8.1 Photography Style

Use photography that feels:

- Real
- Useful
- Modern
- Business-oriented
- Slightly cinematic
- Not generic corporate stock

Best image categories:

- Local businesses
- Service teams
- Owners/operators
- Workshops
- Field technicians
- Coffee/business operations if relevant
- Digital dashboards
- Hands-on work
- Clean workspaces

---

## 8.2 Image Treatment

To place the logo on photography:

1. Add dark overlay if using white/negative logo.
2. Add light overlay if using primary logo.
3. Avoid placing logo over faces, text, or busy texture.
4. Keep logo centered or top-left depending on context.
5. Use monochrome logo when the colored logo competes with the image.

---

## 9. Iconography

Icons should be:

- Simple
- Geometric
- Stroke-based or solid
- Clear at small sizes
- Consistent in line weight

Recommended icon style:

```text
Line icons with 2 px stroke, rounded joins allowed, simple geometric construction.
```

Use accent colors sparingly for icon highlights.

## 10. Brand Voice

## 10.1 Voice Attributes

Thirtyfold Studio should sound:

- Direct
- Practical
- Confident
- Clear
- Business-focused
- Outcome-oriented
- Technically competent
- Human, not robotic

Avoid sounding:

- Overhyped
- Vague
- Too corporate
- Too cute
- Too academic
- Too “AI bro”
- Too spiritual in client-facing sales copy

The biblical meaning can sit quietly under the brand. Client-facing copy should focus on fruit, output, and usefulness.

---

## 10.2 Messaging Principles

### Say This

- “We build AI systems that help you follow up faster.”
- “Turn missed leads into booked conversations.”
- “Automate the work that leaks revenue.”
- “Practical AI agents for real business operations.”
- “More reviews, faster responses, better lead handling.”
- “A studio for AI systems, automation, and business output.”

### Avoid This

- “We revolutionize everything with AI.”
- “Guaranteed 30x growth.”
- “Fully autonomous money machine.”
- “Next-gen paradigm-shifting transformation.”
- “AI magic for your business.”

## 15. Quality Control Checklist

Before approving any logo file, check:

- Is `THIRTYFOLD` spelled correctly?
- Is `STUDIO` spelled correctly?
- Are all letters aligned cleanly?
- Are the colored shadow layers intentional?
- Are there no stray colored nudges?
- Are the letter edges sharp?
- Is the logo centered?
- Is the badge centered under the wordmark?
- Is the transparent background truly transparent?
- Does it work on both light and dark backgrounds?
- Does it remain readable at 180 px wide?
- Is there an SVG master file?
- Are all text elements converted to outlines?

## 19. Final Direction

This logo is strongest as the **primary expressive wordmark** for Thirtyfold Studio. It has recall, energy, and a distinct creative-AI personality.

The critical next step is to rebuild it as a clean vector master file and create a simplified icon system. Without that, the identity will work for social graphics and mockups but will be limited for serious website, print, proposal, and product use.

The final brand system should be:

```text
Bold logo.
Clean layout.
Controlled color.
Practical message.
Measurable output.
```
