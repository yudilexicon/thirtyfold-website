# Thirtyfold Studio — Brand Style Guide

**Version:** 1.0  
**Brand:** Thirtyfold Studio  
**Core visual direction:** Bold creative AI studio, geometric, energetic, professional, high-contrast, visually memorable.

---

## 1. Brand Foundation

### Brand Name

**Thirtyfold Studio**

### Brand Idea

Thirtyfold Studio helps businesses multiply output through AI-powered systems, automation, content, and digital execution. The visual identity should communicate multiplication, momentum, creativity, and technical confidence.

### Brand Personality

Thirtyfold Studio should feel:

- **Sharp** — clean, competent, execution-focused.
- **Creative** — colorful, expressive, visually distinctive.
- **Technical** — structured, systematic, AI-native.
- **Bold** — not generic, not quiet, not corporate beige.
- **Reliable** — professional enough for B2B clients.

### Brand Positioning Statement

**Thirtyfold Studio builds AI-powered systems and creative digital assets that help businesses capture, convert, and compound opportunities.**

### Biblical Anchor

The name “Thirtyfold” carries a multiplication reference: seed that produces a return. The brand should visually express fruitful output, disciplined execution, and multiplied impact without becoming religious in public-facing design unless intentionally used in founder/story content.

## 2. Logo System

### Primary Logo

The primary logo is a custom wordmark:

```text
THIRTYFOLD
  STUDIO
```

The main wordmark uses bold geometric uppercase lettering with hard-edged colorful offset layers. “STUDIO” sits below in a clean rectangular badge.

### Logo Style

The logo style is:

- Bold uppercase geometric wordmark
- Black primary letter face
- White internal separation/highlight edges
- Colorful hard-edged offset shadows
- Retro 3D / pop-art / digital studio aesthetic
- Flat vector-like construction, not glossy or metallic

### Logo Meaning

The layered color shadows represent:

- Multiple outputs from one source
- AI-assisted multiplication
- Creative systems working behind the visible result
- Energy, movement, and compounding execution

### Logo Lockup

The standard lockup is horizontal and stacked:

- **THIRTYFOLD** dominates the mark.
- **STUDIO** is centered below.
- The “STUDIO” badge should never be wider than the main wordmark.

Recommended ratio:

| Element | Approximate Proportion |
|---|---:|
| Main wordmark width | 100% |
| Studio badge width | 24–30% of main wordmark |
| Total logo height | 28–35% of logo width |
| Gap between wordmark and badge | 8–12% of main letter height |

### Clear Space

Use the height of the **S** in “STUDIO” as the minimum clear space around the logo.

Do not place text, icons, borders, or images inside this area.

### Minimum Size

| Use Case | Minimum Width |
|---|---:|
| Website header | 180 px |
| Social post graphic | 240 px |
| Proposal / PDF cover | 300 px |
| Print material | 40 mm |
| Favicon / app icon | Do not use full wordmark |

For anything below 160 px wide, use a simplified icon or monogram instead.

### Logo Variants Required

1. **Primary full-color logo**  
   Black face, white edge separation, colorful offset shadows, black “STUDIO” badge.

2. **Negative full-color logo**  
   White face, darker/adjusted color shadows, white or black “STUDIO” badge depending on background.

3. **Mono black logo**  
   For invoices, legal documents, watermarks, and minimal use.

4. **Mono white logo**  
   For dark backgrounds and video overlays.

5. **Small-use mark**  
   Recommended: `TF`, `30F`, or a simplified layered `T` icon.

### Incorrect Logo Usage

Do not:

- Stretch the logo.
- Add glow effects.
- Add drop shadows behind the full logo.
- Place it on busy images without contrast control.
- Recolor the logo randomly.
- Use low-resolution PNGs for print.
- Use the full wordmark as a tiny favicon.
- Remove the “STUDIO” badge from the primary lockup unless using an approved short mark.

- Orange for conversion

- Cyan/teal/pink for controlled creative accents

## 4. Typography

### Typography Direction

The logo is expressive and heavy, so the supporting typography must be cleaner and more restrained.

Recommended type style:

- Modern sans-serif
- High readability
- Slightly geometric
- Professional B2B feel
- Not playful enough to fight the logo

### Recommended Google Fonts

Use one of these pairings:

#### Option A — Best Overall

| Role | Font |
|---|---|
| Headings | **Space Grotesk** |
| Body | **Inter** |
| UI / Buttons | **Inter** |

This pairing gives a modern AI/startup feel while staying professional.

#### Option B — More Corporate

| Role | Font |
|---|---|
| Headings | **Sora** |
| Body | **Inter** |
| UI / Buttons | **Inter** |

This is cleaner and slightly more SaaS-like.

#### Option C — More Creative

| Role | Font |
|---|---|
| Headings | **Archivo** |
| Body | **Inter** |
| UI / Buttons | **Inter** |

This pairs well with the bold blocky logo.

### Recommended Type Scale

| Element | Desktop Size | Mobile Size | Weight |
|---|---:|---:|---:|
| Hero headline | 64–80 px | 40–48 px | 700–800 |
| Section headline | 40–52 px | 30–36 px | 700 |
| Subheading | 22–28 px | 18–22 px | 500–600 |
| Body | 17–19 px | 16–17 px | 400 |
| Small text | 14–15 px | 13–14 px | 400–500 |
| Button | 15–17 px | 15–16 px | 700 |

### Typography Rules

Do:

- Use tight headlines.
- Use generous line height for body text.
- Keep body text simple and readable.
- Use uppercase sparingly for labels.

Do not:

- Use another decorative display font near the logo.
- Use thin typography for CTAs.
- Use all-caps paragraphs.
- Use more than two type families.

## 5. Layout System

### General Layout Feel

The layout should feel structured, sharp, and conversion-focused.

Recommended design principles:

- Strong grid
- Clear hierarchy
- High contrast
- Modular cards
- Large hero statement
- Bold CTA placement
- Clean visual rhythm
- Controlled use of colorful accents

### Grid

Recommended desktop grid:

- Max content width: **1180–1280 px**
- Columns: **12**
- Gutter: **24 px**
- Section padding: **96–140 px vertical**

Recommended mobile grid:

- Content width: **100% minus 32 px**
- Section padding: **56–80 px vertical**

### Border Radius

| Element | Radius |
|---|---:|
| Buttons | 10–14 px |
| Cards | 18–24 px |
| Small badges | 999 px or 8 px |
| Modal / large panels | 24–32 px |

### Shadows

Use shadows minimally. The logo already has depth.

Recommended card shadow:

```css
box-shadow: 0 18px 60px rgba(17, 17, 17, 0.10);
```

For dark sections:

```css
box-shadow: 0 24px 80px rgba(0, 0, 0, 0.25);
```

## 6. Landing Page Style Direction

### Recommended Landing Page Mood

The landing page should feel like:

**A bold AI systems studio that is creative enough to stand out, but structured enough to be trusted with business operations.**

### Hero Section

Recommended hero structure:

```text
[Logo]
AI systems that turn missed opportunities into booked calls, reviews, and revenue.

Thirtyfold Studio builds automation, AI agents, and conversion systems for service businesses that want faster response, cleaner follow-up, and measurable growth.

[Book a Discovery Call] [See Systems]
```

### Hero Visual Direction

Use one of these:

1. Abstract layered geometric shapes using the palette.
2. Dark indigo dashboard mockup with colorful AI signal lines.
3. Split-screen: business chaos on one side, AI-managed pipeline on the other.
4. Floating cards showing: Voice AI, Lead Reactivation, Review Agent, Web Widget.

Avoid generic robot mascots unless custom-built.

### Section Backgrounds

Recommended sequence:

1. Hero — `#303D67` or white with colorful geometry
2. Problem — `#FFFFFF`
3. Solution cards — `#D7CAC3`
4. Offers — `#FFFFFF`
5. Process — `#303D67`
6. Proof / examples — `#FFFFFF`
7. CTA — `#111111` or `#303D67`
8. Footer — `#111111`

### CTA Buttons

Primary button:

```css
background: #EC9226;
color: #111111;
border-radius: 12px;
font-weight: 700;
padding: 14px 22px;
```

Primary hover:

```css
background: #D37DA7;
color: #111111;
```

Secondary button:

```css
background: transparent;
color: #FFFFFF;
border: 1px solid rgba(255,255,255,0.35);
```

On light backgrounds:

```css
background: transparent;
color: #303D67;
border: 1px solid #303D67;
```

## 7. Graphic Elements

### Accent Shapes

The visual language should borrow from the logo shadows:

- Offset rectangles
- Slanted parallelograms
- Layered blocks
- Hard-edged geometric panels
- Color strips behind headings
- Small 3D-style tabs

### Pattern System

Approved patterns:

- Subtle checker/grid texture
- Offset block pattern
- Thin technical grid lines
- Diagonal color bars
- Abstract basketball-court-inspired geometry from the palette

Use patterns at low opacity, around 4–12%.

### Icon Style

Use icons that are:

- Simple line icons
- Geometric
- 2 px stroke
- Rounded corners acceptable
- One-color or two-color max

Use accent colors only for icon highlights.

### Illustration Style

Approved illustration style:

- Flat vector
- Geometric
- Slight dimensional layering
- Dashboard-like cards
- AI signal lines
- No glossy 3D render style
- No cartoon mascot for core B2B pages

## 8. Photography Direction

### Good Photography Choices

Use photos that feel:

- Real business environments
- Service teams at work
- Clean interiors
- Field/service operations
- People using tools, phones, or laptops
- Warm but professional

### Photo Treatment

To match the palette:

- Add dark indigo overlay at 20–45%.
- Use orange/pink/cyan geometric frames.
- Keep contrast high.
- Avoid over-saturated stock photography.

### Avoid

- Generic AI robot images
- Fake hologram business people
- Overused SaaS stock photos
- Cartoon 3D characters
- Random neon cyberpunk backgrounds

### Cards

Cards should be clean and modular.

Light card:

```css

border: 1px solid rgba(48, 61, 103, 0.12);
border-radius: 24px;
box-shadow: 0 18px 60px rgba(17, 17, 17, 0.08);
```

Dark card:

```css
background: #303D67;
color: #FFFFFF;
border: 1px solid rgba(255,255,255,0.14);
border-radius: 24px;
```

Accent card:

```css
background: #D7CAC3;
color: #111111;
border-radius: 24px;
```

### Badges

Use badges for offer labels and system categories.

Examples:

```text
VOICE AI
LEAD REACTIVATION
GOOGLE REVIEW AGENT
WEB WIDGET
AI FOLLOW-UP
```

Badge style:

```css
background: rgba(25, 184, 242, 0.12);
color: #303D67;
border: 1px solid rgba(25, 184, 242, 0.35);
border-radius: 999px;
font-size: 13px;
font-weight: 700;
letter-spacing: 0.04em;
```

### Forms

Forms should look trustworthy and simple.

```css
input, textarea, select {
  background: #FFFFFF;
  border: 1px solid rgba(48, 61, 103, 0.25);
  border-radius: 12px;
  padding: 14px 16px;
  color: #111111;
}

input:focus, textarea:focus, select:focus {
  border-color: #19B8F2;
  box-shadow: 0 0 0 4px rgba(25, 184, 242, 0.16);
}
```

## 10. Voice & Messaging Style

### Brand Voice

Thirtyfold Studio should sound:

- Clear
- Direct
- Practical
- Confident
- Systems-minded
- Outcome-driven
- Not hype-heavy

### Messaging Principles

Use language that sells outcomes, not tools.

Say:

- “Respond to leads faster.”
- “Reactivate old opportunities.”
- “Turn missed calls into booked appointments.”
- “Automate follow-up without losing the human feel.”
- “Build systems that compound.”

Avoid leading with:

- “We use AI.”
- “We build bots.”
- “We are cutting-edge.”
- “Revolutionary AI transformation.”

### Recommended Taglines

Options:

1. **AI systems that multiply business output.**
2. **Turn missed opportunities into measurable growth.**
3. **Creative AI systems for businesses ready to compound.**
4. **Automation, follow-up, and conversion systems built to multiply.**
5. **From one opportunity to thirtyfold output.**

### Recommended Homepage Headline

```text
AI systems that multiply your follow-up, bookings, and reviews.
```

### Recommended Subheadline

```text
Thirtyfold Studio builds AI agents, automation workflows, and conversion systems for service businesses that need faster response, cleaner follow-up, and measurable growth.
```

## 16. Practical Brand Rules

### The Logo Is Loud. The Website Must Be Disciplined.

Because the logo is colorful and expressive, the landing page should use strong structure and restrained color application.

Best approach:

- Let the logo carry the playful energy.
- Let typography and layout carry trust.
- Let orange carry conversion.
- Let indigo carry authority.
- Let cyan/teal/pink carry AI/creative accents.

### Recommended Visual Balance

Do:

- Use one main CTA color.
- Use dark indigo for serious sections.
- Use the warm neutral to soften the page.
- Use colored geometry as supporting details.

Do not:

- Make the entire website rainbow-colored.
- Use too many gradients.
- Use multiple competing button colors.
- Put the full-color logo on a busy colorful background.

---

## 17. Quality Control Checklist

Before publishing any Thirtyfold Studio asset, check:

- Is the logo sharp and not pixelated?
- Is the logo placed with enough clear space?
- Is the color palette controlled, not chaotic?
- Is orange used consistently for conversion actions?
- Is indigo used to create authority and contrast?
- Is body text readable?
- Are headings strong but not crowded?
- Are accent colors used intentionally?
- Does the page still feel professional, not childish?
- Does the design communicate business outcomes, not just AI aesthetics?

## 18. Final Direction

The Thirtyfold Studio brand should sit between two worlds:

```text
Creative Studio Energy + AI Systems Discipline
```

The logo gives the brand memorability. The palette gives it energy. The layout, typography, and messaging must give it trust.

The winning visual formula is:

```text
Bold black logo face
+ controlled vibrant accents
+ dark indigo authority
+ orange conversion energy
+ clean modern typography
+ structured B2B layout
```

This gives Thirtyfold Studio a distinctive, ownable identity without becoming visually messy.
